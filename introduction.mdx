DRAIT Mini-MES is a web-based Manufacturing Execution System designed specifically for metalworking and welding workshops. It provides complete traceability from quotation to delivery with real-time production management and reporting.

## What is DRAIT Mini-MES?

DRAIT Mini-MES is a full-stack production management system that helps workshop owners, supervisors, and operators manage their entire workflow:

- **Client & Quotation Management** - Create and track quotations from inquiry to approval
- **Work Order Management** - Convert approved quotations to work orders and track them through production
- **Resource Planning** - Manage personnel and equipment assignments
- **Material Tracking** - Monitor inventory with real-time consumption tracking
- **Reporting & Analytics** - Dashboard with KPIs, productivity metrics, and visual charts
- **Audit Trail** - Complete traceability of all operations for compliance

## Technology Stack

DRAIT Mini-MES is built with modern, production-ready technologies:

| Component | Technology |
|-----------|-----------|
| **Frontend** | React 18 + TypeScript + Vite |
| **State Management** | TanStack Query (React Query) v5 |
| **Backend** | NestJS 10 + TypeScript |
| **Database** | PostgreSQL 16 |
| **ORM** | Prisma 5 |
| **Authentication** | JWT + bcrypt |
| **Charts** | Recharts 2 |
| **Infrastructure** | Docker + Docker Compose + Nginx |

## System Requirements

<Note>
DRAIT Mini-MES is designed for on-premise deployment using Docker, making it easy to run on your own infrastructure.
</Note>

### Minimum Requirements

- **Operating System**: Linux, Windows, or macOS with Docker support
- **Docker**: Version 20.10 or higher
- **Docker Compose**: Version 2.0 or higher
- **Memory**: 4GB RAM minimum (8GB recommended)
- **Storage**: 10GB available disk space

### Network Requirements

- Port 80 for HTTP access (configurable)
- Port 5432 for PostgreSQL (internal, not exposed by default)

## Who Should Use DRAIT?

DRAIT Mini-MES is designed for:

- **Small to medium metalworking workshops** looking to digitize their operations
- **Welding and fabrication shops** needing production traceability
- **Workshop owners** who want better visibility into operations and costs
- **Production supervisors** managing work assignments and tracking progress
- **Operators** executing work orders and logging materials
