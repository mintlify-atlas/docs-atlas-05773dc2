## Overview

## GET /api/reports/dashboard

Get comprehensive dashboard data including KPIs and metrics.

**Authentication**: Required (JWT)

**Roles**: All authenticated users (data filtered by role)

### Response

Returns a dashboard object with various metrics and KPIs.

## GET /api/reports/productivity

Get detailed productivity metrics and performance analysis.

**Authentication**: Required (JWT)

**Roles**: All authenticated users

### Query Parameters

<ParamField query="from" type="string">
  Start date for analysis (ISO 8601)
</ParamField>

<ParamField query="to" type="string">
  End date for analysis (ISO 8601)
</ParamField>

<ParamField query="clientId" type="string">
  Filter by specific client
</ParamField>

<ParamField query="operatorId" type="string">
  Filter by specific operator/user
</ParamField>

<ParamField query="status" type="string">
  Filter by work order status
</ParamField>

### Response

Returns detailed productivity metrics.

<ResponseField name="summary" type="object">
  Overall productivity summary
  
  <Expandable title="Summary properties">
    <ResponseField name="totalWorkOrders" type="number">
      Total work orders in period
    </ResponseField>
    
    <ResponseField name="completedWorkOrders" type="number">

    <ResponseField name="totalActualHours" type="number">

    <ResponseField name="avgEfficiency" type="number">

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/api/reports/productivity?from=2026-03-01T00:00:00Z&to=2026-03-31T23:59:59Z',

      'Authorization': `Bearer ${accessToken}`
    }
  }
);

```

```python Python
import requests
from datetime import datetime, timedelta

# Last 30 days
end_date = datetime.now()
start_date = end_date - timedelta(days=30)

response = requests.get(
    'https://api.example.com/api/reports/productivity',
    params={
        'from': start_date.isoformat(),
        'to': end_date.isoformat()
    },
    headers={'Authorization': f'Bearer {access_token}'}
)

productivity = response.json()
for operator in productivity['byOperator']:
    print(f"{operator['operatorName']}: {operator['efficiency']}% efficiency")
```

## Key Metrics Explained

<CardGroup cols={2}>
  <Card title="Efficiency" icon="gauge-high">
    Ratio of estimated time to actual time. Higher is better. Formula: `(estimated / actual) × 100`
  </Card>
  
  <Card title="On-Time Delivery" icon="clock">
    Percentage of work orders delivered by commitment date. Formula: `(onTime / total) × 100`
  </Card>
  
  <Card title="Utilization Rate" icon="chart-pie">
    Percentage of resources actively in use. Formula: `(busy / total) × 100`
  </Card>
  
  <Card title="Conversion Rate" icon="funnel">
    Percentage of quotations converted to work orders. Formula: `(converted / approved) × 100`
  </Card>
</CardGroup>

## Example Use Cases

<Tabs>
  <Tab title="Monthly Performance Review">
    ```javascript
    // Get last month's productivity data
    const lastMonth = new Date();
    lastMonth.setMonth(lastMonth.getMonth() - 1);
    
    const from = new Date(lastMonth.getFullYear(), lastMonth.getMonth(), 1);
    const to = new Date(lastMonth.getFullYear(), lastMonth.getMonth() + 1, 0);
    
    const response = await fetch(
      `https://api.example.com/api/reports/productivity?from=${from.toISOString()}&to=${to.toISOString()}`,
      { headers: { 'Authorization': `Bearer ${token}` } }
    );
    
    const data = await response.json();
    console.log('Monthly efficiency:', data.summary.avgEfficiency);
    ```
  </Tab>
  
  <Tab title="Operator Leaderboard">
    ```javascript
    // Get operator performance for current quarter
    const response = await fetch(
      'https://api.example.com/api/reports/productivity',
      { headers: { 'Authorization': `Bearer ${token}` } }
    );
    
    const data = await response.json();
    const topOperators = data.byOperator
      .sort((a, b) => b.efficiency - a.efficiency)
      .slice(0, 5);
    
    topOperators.forEach((op, i) => {
      console.log(`${i + 1}. ${op.operatorName} - ${op.efficiency}%`);
    });
    ```
  </Tab>
  
  <Tab title="Client Analysis">
    ```javascript
    // Analyze client profitability
    const response = await fetch(
      'https://api.example.com/api/reports/productivity',
      { headers: { 'Authorization': `Bearer ${token}` } }
    );
    
    const data = await response.json();
    const topClients = data.byClient
      .sort((a, b) => b.totalRevenue - a.totalRevenue)
      .slice(0, 10);
    
    topClients.forEach(client => {
      console.log(`${client.clientName}: $${client.totalRevenue} revenue`);
    });
    ```
  </Tab>
</Tabs>

<Note>
  Productivity metrics are calculated based on operation logs and work order timestamps. Ensure operators are consistently logging start, pause, and completion events for accurate reporting.
</Note>
